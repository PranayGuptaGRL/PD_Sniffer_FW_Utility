GRL WinUSB Driver Package
=========================

This folder contains standalone driver packaging assets for the GRL sniffer
device (VID_227F&PID_0005).

Notes
- The package must contain both `grl_sniffer_winusb.inf` and
  `grl_sniffer_winusb.cat`.
- If the sniffer is already connected, unplug and replug it after installation.
